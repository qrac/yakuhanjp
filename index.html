How To Use

DATA

dist/css/yakuhanjp.css
dist/fonts/YakuHanJP/~

HTML

<link rel="stylesheet" href="dist/css/yakuhanjp.css">

CSS

<style>
  * {
    font-family: "YakuHanJP",
    "Hiragino Kaku Gothic ProN", "ヒラギノ角ゴ ProN W3",
    "Meiryo UI", Meiryo, メイリオ, Osaka,
    "ＭＳ Ｐゴシック", "MS PGothic",
    arial, sans-serif;
  }
</style>
